# Smile_Detector
Smile detection program using OpenCV in Jetson Nano(4GB)

OpenCVを使った笑顔検出のプログラム.  
まず顔をがあるかどうかを判定し, 検出された顔が笑顔なら笑い男の画像を合成する.
